# 🌦️ WeatherForYou

> Real-time weather, beautifully analyzed.

WeatherForYou is an interactive weather analytics dashboard built for a hackathon competition at Thiagarajar College of Engineering (TCE), where the project won 🏆

Built using Lovable for vibe coding and rapid development, the platform combines live weather forecasting, analytics, dataset visualization, and a modern futuristic UI experience.

---

# ✨ Features

- 🌍 Live weather search by city
- 📊 Interactive analytics dashboard
- 📈 Real-time weather visualization
- 🌡️ Temperature trend analysis
- 💧 Humidity & pressure tracking
- 🌪️ Wind speed monitoring
- 📁 CSV dataset upload support
- 📉 Beautiful charts & graphs
- 🌌 Modern dark futuristic UI
- 📱 Responsive mobile-friendly design

# 🎨 Design Philosophy

The goal of WeatherForYou was to make weather data feel:
- modern
- interactive
- cinematic
- enjoyable to explore

Instead of creating another boring weather app, the project focuses on:
- immersive UI
- smooth visual hierarchy
- clean analytics presentation
- futuristic aesthetics
